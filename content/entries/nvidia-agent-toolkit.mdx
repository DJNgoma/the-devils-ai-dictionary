plainDefinition: |
  NVIDIA Agent Toolkit is a software toolkit for connecting AI agents to tools, skills, and data sources. Its Omniverse expansion adds callable libraries for sensor simulation, GPU-accelerated physics, and preparing or validating 3D assets for physical-AI simulation.
whyExists: |
  Robotics and industrial simulations require structured scenes with physical properties, sensors, labels, and usable CAD data, not merely attractive images. Tool interfaces let agents help inspect, transform, test, and validate that material inside existing applications.
misuse: |
  Packaging specialised libraries as agent tools is presented as autonomous engineering. The toolkit exposes capabilities; it does not establish that an agent chose the right workflow, produced physically valid assets, or deserves control of the resulting system.
practicalMeaning: |
  Developers can call Omniverse components such as ovrtx, ovphysx, and CAD-to-SimReady skills from agent workflows. They still need schemas, permissions, validation criteria, domain review, provenance, and tests against the real simulator or machine.
