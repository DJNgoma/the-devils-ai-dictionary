  Safetensors is a simple, safe tensor serialisation format designed to store model weights without the arbitrary code execution risks associated with pickle-based formats like some PyTorch defaults historically used.

  Loading weights should not be remote code execution. The format became a hygiene default across much of the open model ecosystem.
