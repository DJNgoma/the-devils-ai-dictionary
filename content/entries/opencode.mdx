plainDefinition: |
  OpenCode is an open-source AI coding agent for terminal, IDE, and desktop workflows, designed to work with multiple model providers and help developers inspect, edit, and operate on codebases.
whyExists: |
  It exists because coding agents moved from autocomplete into delegated repo work, and developers wanted tools that were model-flexible, scriptable, local-friendly, and less tied to a single vendor's product surface.
misuse: |
  It gets used as proof that openness alone creates reliability. An open agent can still edit the wrong file, misunderstand the task, or spend tokens with the confidence of any other agent wearing better governance.
practicalMeaning: |
  In practice it usually means a repo-facing coding agent that can run near a developer's existing workflow, connect to different model providers, and trade polished platform control for flexibility and inspectability.
