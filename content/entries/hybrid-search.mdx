  Hybrid search combines lexical retrieval (for example BM25) with dense vector retrieval, usually via score fusion or reciprocal rank fusion, to improve recall and precision for RAG and search systems.

  Sparse and dense retrievers fail differently. Hybrid systems catch exact identifiers and fuzzy semantics in one pipeline.
