  A web task agent is an AI agent designed to complete end-to-end tasks on websites or web applications, often by browsing, filling forms, extracting information, and taking actions.
whyExists: |
  Many useful workflows span sites, forms, dashboards, and documents without stable APIs. Web task agents try to automate those flows in the same surface a person would use.
misuse: |
  It gets oversold as autonomous web work, when most real web tasks involve fragile state, ambiguous instructions, authentication, recovery, and decisions that should not be hidden behind smooth narration.
practicalMeaning: |
  In practice it means decomposing the task, verifying intermediate state, logging actions, asking for approval at consequential steps, and handling the dull failures that make automation honest.
