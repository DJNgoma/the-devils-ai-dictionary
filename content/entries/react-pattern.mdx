  The ReAct pattern interleaves natural-language reasoning with tool actions and observations in a loop, so an agent can plan and gather information step by step.

  Pure chain-of-thought cannot query systems; pure tool calls lack flexible planning text. ReAct combined both in a widely copied recipe.
