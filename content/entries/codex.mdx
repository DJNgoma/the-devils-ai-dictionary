  Codex is OpenAI's coding agent and coding workflow.   Engineering tasks require repository context, shell and editor tools, isolated environments, tests, diffs, and review—not only generated code. Codex packages those capabilities into local pairing and delegated cloud workflows while the broader ChatGPT product handles additional kinds of work.
