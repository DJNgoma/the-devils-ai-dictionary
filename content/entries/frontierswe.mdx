  FrontierSWE is Proximal's ultra-long-horizon benchmark for coding agents on hard real-world technical challenges. Its tasks span implementation, performance engineering, computational science, and machine-learning research, and are designed to test sustained work near the limits of current human and agent ability.
whyExists: |
  Short issue-resolution benchmarks do not measure whether an agent can research an unfamiliar technical domain, optimise systems, run experiments, recover from failed approaches, and validate a result over several hours. FrontierSWE makes those behaviours central.
misuse: |
  A leaderboard rank is treated as a pure model property even though the harness, tool environment, time limit, cost, stopping behaviour, repeated trials, and task-specific verifier all shape the outcome. Spending longer can reveal persistence or merely extend the mistake.

  Compare results only when the model, harness, task version, tool access, time and token budgets, trial count, correctness gates, and performance scoring match. Inspect trajectories for premature submission, reward hacking, invalid shortcuts, and useful recovery after failed experiments.
