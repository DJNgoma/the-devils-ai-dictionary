  It exists because many real workflows still live behind websites, forms, dashboards, and interfaces that do not expose tidy APIs for every task an agent is asked to complete.

  In practice it means agents can navigate, click, extract, and act inside browser sessions, often with managed infrastructure, proxies, profiles, recordings, and developer APIs around the messy part.
