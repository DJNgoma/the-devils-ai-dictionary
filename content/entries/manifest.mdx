plainDefinition: |
  Manifest is an open-source LLM routing layer for personal AI agents, designed to sit behind OpenAI-compatible endpoints and route requests to cheaper or more suitable models based on task complexity and policy.
whyExists: |
  It exists because many agents send every request to the same large model, even when the task is tiny, repetitive, or cheap enough to deserve a less theatrical engine.
misuse: |
  It gets mistaken for model intelligence rather than routing policy. A router can reduce waste and add fallback behaviour, but it cannot make a weak task definition or poor evaluation strategy suddenly competent.
practicalMeaning: |
  In practice it means model-tiering, fallback chains, budget controls, provider abstraction, local or cloud routing, and a sharper view of what each agent request actually costs.
