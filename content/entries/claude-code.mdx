  A terminal agent for developers who would like to automate the typing without surrendering the blame.
plainDefinition: |
  Claude Code is Anthropic's terminal-oriented coding agent, designed to inspect repositories, edit files, run commands, and assist with multi-step software tasks from the command line.
whyExists: |
  It exists because coding work is not just completion inside an editor. It is shell commands, repo spelunking, test runs, patching, retries, and the general administrative burden of turning intent into a working diff.
misuse: |
  It is oversold as if a terminal agent has somehow escaped the need for software judgement. Faster patch generation is real. So are bad assumptions, missing context, flaky tests, and the delightful persistence of undefined behaviour.
practicalMeaning: |
  In practice it means quicker navigation of unfamiliar codebases, cheaper first drafts of refactors, and more pressure on humans to review command effects instead of admiring how confidently the machine found `package.json`.
example: |
  "Claude Code saved an hour of grep and patching, then presented the team with the ancient obligation to read the diff."
