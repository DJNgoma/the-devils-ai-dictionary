  Project Glasswing is Anthropic's programme for giving selected partners early access to Claude Mythos Preview to find and fix vulnerabilities in critical software and infrastructure.

  Models have become strong enough at code and systems reasoning that cybersecurity is no longer just another benchmark story. If a model can inspect, understand, and alter complex software unusually well, defence and offence both start moving faster.

  It gets treated either as proof that cyber doom is already here or as proof that gated access itself counts as a sufficient safety strategy. Neither conclusion is especially adult. Restricted preview buys time. It does not settle the broader governance problem.

  In practice it means a closed partner programme, usage credits, system cards, research-preview language, and a highly managed attempt to steer frontier capability toward defensive work before the surrounding market catches up in less polite ways.
