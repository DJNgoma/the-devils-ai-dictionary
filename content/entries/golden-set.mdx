  A golden set is a curated evaluation dataset treated as a high-quality reference for testing model or system behaviour.
whyExists: |
  Teams need stable examples that represent important behaviours, edge cases, regressions, and quality expectations. A golden set provides a repeatable target for evaluation.
misuse: |
  It gets treated as objective truth. Golden sets age, encode biases, miss new failure modes, and become dangerous when teams tune to them until the test stops measuring general behaviour.
practicalMeaning: |
  In practice it means careful curation, versioning, contamination controls, holdouts, review cycles, and a refusal to let one spreadsheet become the product's conscience.
