  GPT Image 2 is OpenAI's current image-generation and image-editing model. It accepts text plus image inputs, returns image outputs, and can be used through the Image API or through the Responses API image generation tool.

  Image systems are now expected to do more than spit out pretty approximations. They need stronger prompt adherence, better editing, higher-fidelity preservation of input images, and a smoother path from one-off generation to iterative product workflows.

practicalMeaning: |
  In practice it means OpenAI's latest image model for generation and edits, with flexible sizing and stronger image-input handling. The operational wrinkle is that the model itself sits behind two different patterns: direct image endpoints such as `v1/images/generations` and `v1/images/edits`, or a mainline model in the Responses API using the `image_generation` tool for multi-turn editing and prompt revision.
