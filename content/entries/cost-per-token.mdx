  Cost per token is the price charged or estimated for processing or generating a token of text (or multimodal equivalent), usually quoted separately for input and output and used as a rough unit economics measure.

  Usage-based model APIs needed a billable unit finer than seats or servers. Tokens became the common currency of inference spend.

  Compared across vendors without matching context length, output ratio, caching, batch discounts, or quality. A cheaper token that forces more retries is not cheaper.

  Model total cost as input tokens, output tokens, cached tokens, tool calls, and failure retries. Track cost per successful task, not only list price per million tokens.
