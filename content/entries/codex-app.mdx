  The ChatGPT desktop app is OpenAI's current desktop surface for conversation, Work mode, artefact creation, connected applications, and the Codex coding experience.   OpenAI combined its desktop coding-agent experience with a broader agent that can work across applications and create documents, spreadsheets, slides, and other deliverables. One shell reduces the boundary between coding and general computer-based work.
