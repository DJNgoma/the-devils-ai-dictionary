devilDefinition: |
  Moonshot's name for the assistant, the workspace, the coding tools, and the model family, because one noun is cheaper than four onboarding screens.
plainDefinition: |
  Kimi is Moonshot AI's consumer and developer product brand. It covers an agentic AI workspace, Kimi Work, Kimi Code, web and app experiences, an API platform, and a changing family of Kimi models; the name does not identify one fixed model or capability.
whyExists: |
  Moonshot uses one product family for general assistance, research, knowledge work, presentations, coding, long-running tasks, and model access. A shared brand lets the product combine specialised interfaces and agent workflows without asking users to assemble them independently.
misuse: |
  People attribute a result to "Kimi" without recording whether it came from Kimi K3, an older Kimi model, Work, Code, a swarm or goal workflow, or an API integration. Product scaffolding and model capability then become one flattering but unrepeatable benchmark.
practicalMeaning: |
  Buyers and evaluators should name the active model, surface, tools, effort setting, connected data, and execution mode. Kimi.com, Kimi Work, Kimi Code, and the API can expose different affordances even when they share a model family.
