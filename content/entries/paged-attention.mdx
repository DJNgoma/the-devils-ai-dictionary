plainDefinition: |
  PagedAttention is a KV-cache management technique (popularised by vLLM) that stores attention key/value tensors in non-contiguous pages, improving memory utilisation for batched LLM serving.
whyExists: |
  Request lengths vary; contiguous preallocation wastes memory. Paging lets servers admit more concurrent sequences.
misuse: |
  Cited as automatic infinite context. It improves packing efficiency; it does not remove model context limits.
practicalMeaning: |
  Use serving engines that implement it well, monitor fragmentation and preemption, and size pages for your length distribution.
