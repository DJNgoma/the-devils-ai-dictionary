devilDefinition: |
  The feature where a model is allowed to click around software with the confidence of a junior employee and the supervision requirements of a forklift.
plainDefinition: |
  "Computer use" refers to AI systems that can operate graphical user interfaces by observing screens, deciding actions, and interacting through clicks, typing, scrolling, or other desktop and browser controls.
whyExists: |
  It exists because many real workflows still live inside user interfaces rather than neat APIs. If a system can only answer questions but cannot operate the tools where work actually happens, its usefulness in operations remains politely theoretical.
misuse: |
  It is oversold as if seeing a screen and moving a cursor were the same as understanding business process, edge cases, or risk. A demo that closes one support ticket can be described as labour disruption by lunch.
practicalMeaning: |
  In practice it means brittle automation with vision, planning, retries, guardrails, and a strong need for scoped permissions. It can save time on repetitive interface work, provided someone is willing to own the mistakes made at machine speed.
