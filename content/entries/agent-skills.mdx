devilDefinition: |
  A way to give an agent specialised habits without shoving the entire employee handbook into every prompt.
plainDefinition: |
  Agent skills are reusable bundles of instructions, metadata, scripts, templates, and reference material that an AI agent can load when a task calls for a particular capability.
whyExists: |
  They exist because agents need domain procedures and repeatable workflows, but putting every possible instruction into the main context window is expensive, noisy, and a reliable way to teach the model nothing in particular.
misuse: |
  They get treated as plugins, permissions, or magic competence capsules. A skill can guide behaviour and provide tools, but it still needs trust boundaries, review, versioning, and a system that knows when not to use it.
practicalMeaning: |
  In practice it means packaging domain knowledge so the agent can discover the skill cheaply, load detailed guidance only when needed, and run deterministic helper scripts or consult reference files without bloating every task.
example: |
  "The finance skill helped the agent produce the right spreadsheet; the security review then asked why everyone had installed it with the serenity of a gift basket."
