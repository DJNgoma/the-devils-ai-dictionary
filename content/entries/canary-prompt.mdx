  A canary prompt is a stable test input (or small set) run continuously or at deploy time to detect behavioural regressions, configuration mistakes, or unexpected model swaps.

  LLM systems drift via prompts, models, tools, and decoding settings. Canaries provide cheap smoke tests in production.

  Maintain a diverse canary pack, alert on unexpected deltas, and keep expected outputs or graded properties versioned beside the prompts.
