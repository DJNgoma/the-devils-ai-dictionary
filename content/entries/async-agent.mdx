  An async agent runs workflows that do not require a continuous interactive session; results return later through notifications, pull requests, tickets, email, or dashboards.

  Not all valuable work fits a chat bubble. Product surfaces needed agents that integrate with existing async collaboration tools.

  Define completion contracts, status updates, failure notifications, and idempotency. Pair async agents with traces humans can open hours later.
