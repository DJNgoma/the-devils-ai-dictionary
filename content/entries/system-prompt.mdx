  The invisible preface explaining what the machine is supposed to be before the user begins testing the claim.
plainDefinition: |
  A system prompt is the high-priority instruction layer supplied by the application or developer to shape a model's behaviour, rules, tone, and operating constraints.
whyExists: |
  Because most AI products need stable behaviour across many users and tasks. Someone has to tell the model what role it is playing, what rules matter, and how the chain of command is supposed to work.
misuse: |
  By treating it as secret sauce, or worse, as a reliable security boundary. A system prompt is useful control text. It is not a magical wall separating trusted intent from hostile input.
practicalMeaning: |
  In practice it usually means a standing instruction layer covering role, tone, refusal rules, output format, tool permissions, escalation behaviour, and the product's preferred version of reality.

  "The app felt polished until the leaked system prompt revealed that half the brand voice was an anxious paragraph about never improvising legal advice."
